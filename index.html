07

The risk & the fix watch

⚠ The trap

Corrie's strawman: slide 2 = non-profit, profits back to JMJ; slide 9 = SodaLabs as a vendor cost-line next to Azure/Cisco. The deck specs an engine we've already built. "Greater-good wrappers become reasons to never pay." Pour the engine into a non-profit with no structural return → we fund it all and earn nothing.

✓ The fix

Engine licensed, never assigned. SodaLabs = founding technology owner-partner with the licence fee + per-city model living outside / above the non-profit. Get it in writing while it's still a strawman — the cheapest moment to set terms. (Team slide already says "constituted as CV and MB" — if MB = Mic, we negotiate from inside-out.)
